# All-in-1 Software 🚀

A complete desktop application with multiple useful tools!

## Features ✨

1. **🎨 Paint** - Draw and paint with different colors and pen thicknesses
- Choose a color
- Adjust pen thickness
- Save drawing as PNG
- Clear canvas

2. **📝 Text Editor** - Full-featured text editor
- Open and save files
- Adjust font and size
- Multiple fonts available

3. **🔢 Calculator** - Advanced calculator
- Basic arithmetic functions (+, -, *, /)
- Brackets support
- Decimal numbers
- Backspace function

4. **📋 Notes** - Quick note-taking
- Add notes
- Delete notes
- Overview list

## Installation 💻

### Requirements
- Python 3.7 or higher
- pip (Python package installer)

### Step 1: Install required packages

```bash
pip install pillow
```

> **Note:** tkinter comes with Python by default, but if you get an error:
> - **Ubuntu/Debian**: `sudo apt-get install python3-tk`
> - **macOS**: Comes with Python by default
> - **Windows**: Comes with Python by default

### Step 2: Start the program

```bash
python all_in_one.py
```

Or on some systems:

```bash
python3 all_in_one.py
```

## Use 📖

### Paint
1. Select the Paint tab
2. Choose a color with the "Color" button
3. Adjust the pen thickness with the slider
4. Draw on the white canvas with your mouse
5. Save your artwork with "Save"

### Text Editor
1. Select the Text Editor tab
2. Start typing or open an existing file
3. Adjust the font and size as desired
4. Save your Save the document with "Save"

### Calculator
1. Select the Calculator tab
2. Click the buttons to enter a calculation
3. Press "=" to see the result
4. Use "C" to delete and "←" to remove the last digit

### Notes
1. Select the Notes tab
2. Click "New Note" and enter your note
3. Select a note and click "Delete Selected" to delete it

## System Requirements ⚙️

- **Operating System**: Windows, macOS, or Linux
- **RAM**: Minimum 512 MB
- **Disk Space**: 50 MB of free space

## Troubleshooting 🔧

**Program won't start:**
- Check that Python is installed correctly: `python --version`
- Check that Pillow is installed: `pip list | grep -i pillow

**Cannot save files:**
- Make sure you have write permissions in the folder you're trying to save to.

**Interface looks strange:**
- Try resizing the window.
- Restart the program.

## Tips 💡

- You can switch between tabs with your mouse.
- Use keyboard shortcuts in the text editor (Ctrl+C, Ctrl+V, etc.).
- Paint drawings are automatically saved as PNGs for the best quality.
- The calculator supports complex calculations such as: (5 + 3) * 2.
